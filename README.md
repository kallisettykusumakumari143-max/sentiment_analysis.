Sentiment Analysis Using Machine Learning
1.Project Overview

Sentiment Analysis is a Natural Language Processing (NLP) and Machine Learning project that analyzes text and determines whether the sentiment expressed is positive or negative. This project helps computers understand human opinions, emotions, and feedback from text data.
The system is trained using sample text data and sentiment labels. It converts text into numerical features using CountVectorizer and applies the Multinomial Naive Bayes algorithm to classify the sentiment of new text inputs.

2.Project Description
The main objective of this project is to build a simple sentiment classification model that can automatically identify whether a given sentence expresses a positive or negative opinion.

3.The project uses:
Python programming language
Scikit-learn library
CountVectorizer for text feature extraction
Multinomial Naive Bayes for classification
The model is trained on labeled text samples and predicts the sentiment of new user-provided sentences.

4.Objectives
To understand the basics of Machine Learning and NLP.
To classify text into Positive or Negative sentiment.
To automate sentiment detection from textual data.
To gain practical experience with Python and Scikit-learn.
Technologies Used
Python
Scikit-learn
CountVectorizer
Multinomial Naive Bayes
Visual Studio Code (VS Code)

5.Working Process
Collect text data with sentiment labels.
Convert text into numerical features using CountVectorizer.
Train the Machine Learning model.
Input a new sentence.
Predict whether the sentiment is Positive or Negative.
Display the prediction result.
